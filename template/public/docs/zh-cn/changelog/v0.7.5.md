# v0.7.5

## 修复

- 修复 `generate-doc-git-meta` 在 docs 项目不位于 git 根目录时无法生成最近更新时间的问题。
- 修复 `generate-changelog-index`、`generate-llms-files`、`generate-search-index` 对 Windows `CRLF` frontmatter 解析异常的问题。
- 修复 changelog 摘要、页面元信息和 LLM 导出内容错误包含 frontmatter 原文的问题。

## 兼容性

- 模板脚本现在统一兼容 `LF`、`CRLF` 和 BOM 文本头部。
