# v0.8.0

## 新增

- 模板构建流程集成 `react-docs-ui generate-sitemap` 和 `react-docs-ui generate-feed`，构建时自动生成 `sitemap.xml` 和 `feed.xml`。
- 模板 `site.yaml` 新增 `sitemap`、`feed`、`breadcrumb`、`reading` 配置节，支持一键开启/关闭各项功能。

## 配置变更

- `site.yaml` 新增 `sitemap` 配置（`enabled`、`changefreq`、`priority`、`exclude`）。
- `site.yaml` 新增 `feed` 配置（`enabled`、`title`、`description`、`limit`）。
- `site.yaml` 新增 `breadcrumb` 配置（`enabled`、`showHome`）。
- `site.yaml` 新增 `reading` 配置（`showTime` 默认开启、`showProgress` 默认关闭）。

## 工程

- `predev` / `prebuild` 脚本新增 `generate-sitemap` 和 `generate-feed` 子命令。
- 中英文 `site.yaml` / `site.en.yaml` 同步更新所有新增配置项。
