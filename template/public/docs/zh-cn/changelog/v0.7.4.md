# v0.7.4

## 修复

- 修复 Windows 环境下 `CRLF` 换行的 Markdown 和 MDX 文档无法正确剥离 frontmatter 的问题。
- 修复使用 `create-react-docs-ui` 创建的模板项目中，页面头部元数据被直接显示在正文里的问题。

## 兼容性

- frontmatter 解析现在同时兼容 `LF`、`CRLF` 和带 BOM 的文档内容。
