# v0.7.3

## 修复

- 修复 template 项目中 `Alert`、`Feature`、`Callout`、`BadgeList`、`MyTip` 等自定义组件无法在 MDX 中识别的问题。
- 修复自定义组件在本地联调与发布构建两种入口模式下的注册链路。

## 工程

- 为 template 拆分本地联调与发布构建两套启动文件。
- 为 `create-react-docs-ui` 增加发布前自动切换模板入口的脚本。
- 补齐发布包所需的启动辅助文件。
