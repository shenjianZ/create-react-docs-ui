# v0.9.4

## 依赖

- `react-docs-ui` 依赖升级到 `0.9.4`，Mermaid 图表现在支持全屏缩放查看，大图文字不再被压缩。
- 模板字体发布清单补齐 `Noto Sans SC` 与 `MiSans` 字体，生成项目不会再因猜测字体路径命中 Vite fallback HTML 而出现字体解码警告。

## 文档

- 更新 `mdx-test` 示例页，新增节点较多的大图与时序图示例，用于演示全屏查看交互。

## 工程

- 脚手架版本升级到 `0.9.4`，用于替代已撤回的 `0.9.3`。
