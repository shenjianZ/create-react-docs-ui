# v0.7.6

## 新增

- 模板项目接入 `react-docs-ui` 集中化 CLI，构建脚本通过 `react-docs-ui generate-*` 子命令调用。
- 支持本地联调模式，模板通过 `file:` 协议引用本地 `react-docs-ui` 源码进行开发。

## 修复

- 修复脚手架创建项目时未正确检测用户包管理器（pnpm / yarn / npm）的问题。
- 修复模板文档本地联调工作流中路径解析异常的问题。

## 工程

- 模板 `package.json` 中 `predev` / `prebuild` 脚本迁移到 `react-docs-ui` CLI 子命令模式。
