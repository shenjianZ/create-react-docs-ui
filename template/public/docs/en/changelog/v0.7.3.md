# v0.7.3

## Fixed

- Fixed custom MDX components such as `Alert`, `Feature`, `Callout`, `BadgeList`, and `MyTip` not being resolved in the template project.
- Fixed the registration flow for custom components across both local-link development and published-package startup modes.

## Engineering

- Split the template startup helpers into local-link and published-package variants.
- Added a prepack switch script so `create-react-docs-ui` publishes the correct template entry.
- Included the required startup helper files in the published package.
