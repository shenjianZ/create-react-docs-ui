# v0.7.6

## Added

- Template projects now use the centralized `react-docs-ui` CLI; build scripts are invoked via `react-docs-ui generate-*` subcommands.
- Local development mode supported — templates can reference local `react-docs-ui` source via `file:` protocol.

## Fixed

- Fixed scaffold not correctly detecting the user's package manager (pnpm / yarn / npm) during project creation.
- Fixed path resolution issues in the template's local development workflow.

## Engineering

- Migrated template `package.json` `predev` / `prebuild` scripts to `react-docs-ui` CLI subcommand mode.
