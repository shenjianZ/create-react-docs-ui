# v0.7.4

## Fixed

- Fixed Markdown and MDX files with Windows `CRLF` line endings not stripping frontmatter correctly.
- Fixed template projects created by `create-react-docs-ui` showing page metadata directly in the document body.

## Compatibility

- Frontmatter parsing now supports `LF`, `CRLF`, and BOM-prefixed document content.
