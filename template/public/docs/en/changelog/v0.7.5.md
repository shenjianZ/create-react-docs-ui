# v0.7.5

## Fixed

- Fixed `generate-doc-git-meta` so template projects can resolve last-updated metadata even when the docs app is inside a git subdirectory.
- Fixed `generate-changelog-index`, `generate-llms-files`, and `generate-search-index` to parse Windows `CRLF` frontmatter correctly.
- Fixed changelog summaries, page metadata, and LLM exports accidentally including raw frontmatter text.

## Compatibility

- Template scripts now consistently support `LF`, `CRLF`, and BOM-prefixed document content.
