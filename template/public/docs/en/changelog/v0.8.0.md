# v0.8.0

## Added

- Template build pipeline now integrates `react-docs-ui generate-sitemap` and `react-docs-ui generate-feed`, automatically producing `sitemap.xml` and `feed.xml` at build time.
- Template `site.yaml` adds `sitemap`, `feed`, `breadcrumb`, and `reading` config sections for toggling each feature.

## Configuration Changes

- `site.yaml` adds `sitemap` config (`enabled`, `changefreq`, `priority`, `exclude`).
- `site.yaml` adds `feed` config (`enabled`, `title`, `description`, `limit`).
- `site.yaml` adds `breadcrumb` config (`enabled`, `showHome`).
- `site.yaml` adds `reading` config (`showTime` default: enabled, `showProgress` default: disabled).

## Engineering

- `predev` / `prebuild` scripts now include `generate-sitemap` and `generate-feed` subcommands.
- Both `site.yaml` and `site.en.yaml` updated with all new config sections.
