# v0.9.4

## Dependencies

- Bumped `react-docs-ui` to `0.9.4`; Mermaid diagrams now support a fullscreen zoom viewer and large diagrams are no longer shrunk to unreadable text.
- Included the full `Noto Sans SC` and `MiSans` font set in the template publish list, preventing generated sites from hitting Vite fallback HTML for guessed font paths and logging font decode warnings.

## Documentation

- Updated the `mdx-test` example page with larger flowchart and sequence diagrams to demonstrate the fullscreen viewer.

## Engineering

- Bumped the scaffold version to `0.9.4` to replace the withdrawn `0.9.3` release.
